亚森罗“苹”/ by 安斯拉托利斯克

以一个幻想世界为舞台的轻度日常推理作品

// 一 // // 二 //

一 ：【二次下注】

    粟礼中学有着和其他学校不一样的校园祭，校园节总在考完本学年的期中考后， 留下午来发新书，办杂事。隔天，便是由老师们和早就准备好的学生们开展校园节， 给学生聊以慰藉。

    今天则是校园节的前一天，刚考完英语， 有些晕头转向，也许是这几天过度用脑了。我尽力走出考场教室， 在去往二年A班教室途中，路过了告示牌。

    “啊……”

    我看到告示栏上有着【号外！！】 的粟礼中学版报社的报道。粟高的版报社的内容总是例行公事和鸡毛蒜皮， 社员也非常清楚吧，所以这等事也被这样重视地被批上号外。

——————————
    【号外！！】
    校园节上将会有突发事件！
    本社今早在收取信息收集箱时， 竟收到署名为【亚森 罗苹】的来信！信的内容大致如下：

同学们：
    致以考后的问候。
    校园节真是个美好的节日不是吗？ 学生们在紧张的考试后还能在放假前享受一番， 那在如此快乐的节日里，我决定将气氛再炒热些。
    我将会在届时开一些玩笑，与大伙玩找不同。
    当然是有规律的，与灰姑娘一样，在“钟声”敲响之际， 也许某处的物品已然更替。
    使用的是上学期学务会省吃俭用下来的经费。
    让我们都享受这次狂欢吧，我期待大众的表现。
    亚森 罗苹
——————————

    亚森罗苹？我记得那是法国人写的一个著名怪盗形象，但我看的版本翻译的是“罗平”， 兴许是那位“绅士怪盗”想要营造一些幽默感。不过也许是大多数人连亚森罗平都不知道吧。

    粟礼中学学务会今年的活是这样的吗？每年都办的， 上次是把游园会加入了追逐系统来着。

    我关于这种突发事件，确实没有什么好的记忆。

    话说这也算例行公事了，毕竟每年都办校园节。

    我平静地走到座位上，望着外头操场上准备着工作的学生们， 多半是学务会的吧？其中有一人是我认识的，西尾维人，我们在一年级是同桌。

    视线移回来，二年的课本被逐本置于我的课桌上， 比我想象中要多。发书的学生向我搭话：“这次也是怪盗哦，野坂你打算怎么办？”

    我抬头才发现是相叶，这位倒是认识挺久了， 她从初中就开始跟着我了，不是我故意用这种自作多情的词，而是我们都有一样的心结。

    “啊，是啊，去年我可输惨了， 明明名号用的都是比这次更大众话的怪盗，‘基德’、‘十文字’什么的，我不甘心。”

    是的，若是心服口服我也不会有这种想法了。

    在上次，我使尽全身解数却还是没能做到，施展我的才能， 让西尾维人成功解出了谜题。

    我的自信被击碎了。

    但是，我抱着最后一点希望，给我的才能证明一次。
